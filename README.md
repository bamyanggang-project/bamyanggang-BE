## 6. 역할분담 

###  🍊팀장 김동규(FE,BE)
- **UI**
  - 마이페이지 서브 (회원가입, 회원수정, 로그인, 로그아웃, 회원탈퇴)

  - 유효성 검사를 활용한 회원가입 (CRUD)

  - Docker + AWS + nginx 배포
